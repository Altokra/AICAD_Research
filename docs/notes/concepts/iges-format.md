# IGES Format

## 定义

IGES (Initial Graphics Exchange Specification) 是 CAD 数据交换的**老标准格式**，1970s 由 NIST 开发。用于在不同 CAD 系统之间交换几何数据。

## 直觉理解

在 STEP 出现之前，IGES 是 CAD 软件之间的"翻译官"。虽然老了，但很多老系统中仍有 IGES 文件。

## 文件结构

IGES 文件分为 5 部分（每行 80 字符）：

```
┌──────────────────────────────────────────────────────────────┐
│  START (第1行)          │  描述信息，通常为空或注释         │
├──────────────────────────────────────────────────────────────┤
│  GLOBAL               │  前言信息                           │
│                       │  参数分隔符、版本、单位等           │
├──────────────────────────────────────────────────────────────┤
│  DIRECTORY ENTRY (DE)  │  每个实体一条记录（80字符）        │
│                       │  存储实体类型、指针等信息          │
├──────────────────────────────────────────────────────────────┤
│  PARAMETER DATA (PD)   │  实体的实际参数数据               │
│                       │  存储坐标、控制点等                │
├──────────────────────────────────────────────────────────────┤
│  TERMINATE (最后一行)   │  文件结束标记                     │
└──────────────────────────────────────────────────────────────┘
```

## 支持的实体类型

### 几何实体
| 编号 | 实体 | 说明 |
|------|------|------|
| 100 | Circular Arc | 圆弧 |
| 104 | Conic Arc | 圆锥曲线 |
| 106 | Copious Data | 曲线/曲面数据点 |
| 110 | Line | 直线 |
| 112 | Parametric Spline | 参数样条曲线 |
| 114 | Parametric Surface | 参数曲面 |
| 120 | NURBS Curve | NURBS 曲线 |
| 128 | NURBS Surface | NURBS 曲面 |

### 结构实体
| 编号 | 实体 | 说明 |
|------|------|------|
| 410 | Color | 颜色定义 |
| 502 | Block | 块（几何体） |

## IGES 的问题

### 1. 数据丢失
- 颜色、图层、材料属性经常丢失
- 复杂曲面转换不准确
- 精度有限

### 2. 版本混乱
- 不同软件支持不同 IGES 版本
- 有些软件导出 IGES 格式不规范

### 3. 文件膨胀
- 文本格式，重复数据多
- 很大文件难以处理

## IGES vs STEP

| 维度 | IGES | STEP |
|------|------|------|
| **年龄** | 老 (1970s) | 新 (1990s+) |
| **标准** | NIST (非 ISO) | ISO 国际标准 |
| **几何支持** | NURBS、曲面 | 更完整的 B-Rep |
| **精度** | 中等 | 更高 |
| **兼容性** | 差（软件差异大） | 好 |
| **现代 CAD** | 逐渐淘汰 | 主流 |

## Python 读写

### OpenCASCADE
```python
from OCP.IGESControl import IGESControl_Reader, IGESControl_Writer

# 读取
reader = IGESControl_Reader()
reader.ReadFile("model.igs")
reader.TransferRoots()
shape = reader.OneShape()

# 写入
writer = IGESControl_Writer()
writer.AddShape(shape)
writer.Write("output.igs")
```

### 其他库
- **pythonocc**：OpenCASCADE Python 绑定
- **cadquery**：支持 IGES 导入

## 何时用 IGES vs STEP

| 场景 | 推荐格式 |
|------|----------|
| 现代 CAD 软件之间 | STEP |
| 老系统 / 遗留数据 | IGES |
| 只需要曲面外形 | 两者皆可 |
| 需要完整实体信息 | STEP |
| 3D 打印 | STL（网格） |
