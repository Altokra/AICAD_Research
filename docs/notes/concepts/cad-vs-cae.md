# CAD vs CAE

## 定义

### CAD (Computer-Aided Design)
- **计算机辅助设计**
- 创建几何模型
- 回答"这个零件长什么样"

### CAE (Computer-Aided Engineering)
- **计算机辅助工程**
- 分析几何模型的物理性能
- 回答"这个零件能不能用"

## 对比

| 维度 | CAD | CAE |
|------|-----|-----|
| **目的** | 创建设计 | 分析性能 |
| **输出** | 几何模型 | 应力/温度/流场等 |
| **关注** | 形状 | 物理行为 |
| **软件** | SolidWorks, CATIA | ANSYS, Abaqus, Nastran |

## CAE 分析类型

### 结构分析 (Structural Analysis)
- 静力学 (Static)
- 动力学 (Dynamic)
- 模态分析 (Modal)
- 屈曲分析 (Buckling)

### 热分析 (Thermal Analysis)
- 稳态热
- 瞬态热

### 流体力学 (CFD)
- 流体仿真
- 风洞模拟

### 多物理场
- 热-结构耦合
- 流-固耦合

## 工程流程

```
     CAD 建模                    CAE 分析
       ↓                           ↓
┌─────────────┐            ┌─────────────┐
│ 创建几何    │     →      │ 前处理      │
│ 零件/装配   │            │ 划分网格    │
└─────────────┘            │ 设置边界条件 │
                            │ 设置材料    │
                            └─────────────┘
                                    ↓
                            ┌─────────────┐
                            │ 求解        │
                            │ 调用求解器  │
                            └─────────────┘
                                    ↓
                            ┌─────────────┐
                            │ 后处理      │
                            │ 查看结果    │
                            │ 云图/动画  │
                            └─────────────┘
```

## 网格划分 (Meshing)

CAE 分析的核心步骤：

1. **几何导入**：从 CAD 读取几何
2. **网格生成**：将几何离散成小单元
3. **单元类型**：
   - 2D：三角形、四边形
   - 3D：四面体、六面体、金字塔

### 网格质量
- **单元尺寸**：越小越精确，但计算量越大
- **纵横比**：避免畸形单元
- **雅可比值**：衡量单元质量

## 水密性的关键作用

### 问题来源

```
CAD 模型                    CAE 要求
┌─────────┐                ┌─────────┐
│ 零件    │─────导入────→  │ 网格    │
└─────────┘                └─────────┘
                              ↓
                         必须水密！
                         不能有孔洞/间隙
```

### 水密性要求
- **封闭表面**：每个边恰好被两个面共享
- **无自交**：面片不能和自己交叉
- **正确法向**：所有法线方向一致

### 如果不水密？
- 网格划分失败
- 仿真结果不准确
- 无法进行流体分析（漏水）

## 实际案例：注塑件分析

```
1. CAD 建模（Creo/SolidWorks）
   ↓ 设计塑料壳体

2. 导出 STEP 文件
   ↓ 几何信息完整

3. 导入 ANSYS
   ↓

4. 前处理
   - 设置材料（塑料）
   - 划分网格（网格质量检查）
   - 施加约束和载荷

5. 求解
   - 静力学分析
   - 查看应力分布

6. 后处理
   - 云图可视化
   - 安全系数评估
```

## CAD 修复的位置

```
CAD 模型 ──→ CAD 修复 ──→ CAE 分析
  ↓              ↓            ↓
可能有缺陷     修复水密性     成功划分网格
                             精确仿真
```

## 为什么这是研究热点？

1. **数据转换丢失**：CAD 格式转换（STEP → 中间格式）可能丢失几何信息
2. **设计缺陷**：设计师不了解 CAE 要求
3. **自动化需求**：希望 AI 自动检测和修复问题
4. **深度学习应用**：用神经网络学习修复策略
