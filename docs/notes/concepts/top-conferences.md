# Top Conferences in CV and AI

## 计算机视觉 (Computer Vision)

### CVPR - Computer Vision and Pattern Recognition
- **级别**：Top (CCF-A)
- **主办**：IEEE
- **时间**：每年 6 月
- **地点**：北美
- **录取率**：约 25-30%
- **影响力**：CV 领域最高
- **官网**：cvpr.cc

**涉及方向**：
- 3D Vision / Point Cloud
- Object Detection / Segmentation
- Image Generation / Diffusion
- Neural Rendering / NeRF
- Video Understanding

### ICCV - International Conference on Computer Vision
- **级别**：Top (CCF-A)
- **主办**：IEEE
- **时间**：奇数年
- **地点**：全球
- **录取率**：约 20-25%
- **影响力**：与 CVPR 齐名
- **官网**：iccv.cc

### ECCV - European Conference on Computer Vision
- **级别**：Top (CCF-A)
- **主办**：Springer
- **时间**：偶数年
- **地点**：欧洲
- **官网**：eccv.ec

## 机器学习 / AI (Machine Learning / AI)

### NeurIPS - Neural Information Processing Systems
- **级别**：Top (CCF-A)
- **主办**：NeurIPS Foundation
- **时间**：每年 12 月
- **地点**：北美
- **录取率**：约 26%
- **影响力**：AI/ML 最高
- **官网**：neurips.cc

**涉及方向**：
- Deep Learning Theory
- Optimization
- Reinforcement Learning
- Generative Models (GAN, Diffusion)
- NLP / Transformers
- Graph Neural Networks

### ICLR - International Conference on Learning Representations
- **级别**：Top (CCF-A)
- **主办**：OpenReview
- **时间**：每年 4 月
- **地点**：全球
- **录取率**：约 30%
- **影响力**：深度学习顶会
- **官网**：iclr.cc

**涉及方向**：
- Representation Learning
- Self-supervised Learning
- Transformers
- Diffusion Models
- AI for Science

### ICML - International Conference on Machine Learning
- **级别**：Top (CCF-A)
- **主办**：ICML
- **时间**：每年 7-8 月
- **官网**：icml.cc

## 如何跟踪最新论文

### 1. arXiv
- **地址**：https://arxiv.org
- **关注领域**：cs.CV, cs.LG, cs.GR
- **每日更新**

### 2. Papers with Code
- **地址**：https://paperswithcode.com
- 论文 + 代码 + 排行榜

### 3. Connected Papers
- **地址**：https://connectedpapers.com
- 论文引用图谱

### 4. Google Scholar Alerts
- 设置关键词提醒
- 当有新论文时邮件通知

### 5. Twitter / X
- 关注领域研究者
- 很多论文在这里首发

### 6. Reading Groups
- 实验室组会分享
- Discord/Slack 学术社区

## 建议：如何高效读论文

### 1. 粗筛 (5 分钟)
- 标题 + 摘要 + 图
- 判断是否相关

### 2. 泛读 (15-20 分钟)
- 问题是什么
- 方法是什么
- 结果好不好

### 3. 精读 (1-2 小时)
- 仔细看方法
- 复现核心思想
- 写笔记

### 4. 代码复现 (可选)
- 跑通官方代码
- 改进或应用
