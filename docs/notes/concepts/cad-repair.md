# CAD Repair

## 定义

CAD 修复指对存在几何缺陷（孔洞、间隙、自交等）的 CAD 模型进行自动或半自动修正，使其满足工程使用要求（通常是水密性）。

## 为什么需要 CAD 修复？

### 1. 数据转换丢失
```
SolidWorks (原生格式) ──→ STEP ──→ 另一软件
                                  ↓
                            可能丢失信息
                                  ↓
                            模型有缺陷
```

### 2. 逆向工程
```
扫描数据 ──→ 点云 ──→ 网格重建
                    ↓
              重建可能有缺陷
```

### 3. AI 生成
```
AI 模型生成 ──→ 可能有孔洞/间隙
```

### 4. 老旧数据
```
旧 CAD 文件 ──→ 格式老旧 ──→ 质量问题
```

## 传统修复方法

### 1. 几何修复 (Geometry Repair)

| 操作 | 说明 |
|------|------|
| **顶点合并** | 合并距离过近的顶点 |
| **边缝合** | 缝合间隙 |
| **面片填补** | 填充孔洞 |
| **法线重定向** | 统一法线方向 |
| **去除重复面** | 移除重叠面 |

### 2. 网格处理

| 操作 | 说明 |
|------|------|
| **重网格化 (Remeshing)** | 重新生成高质量网格 |
| **细分 (Subdivision)** | 平滑网格 |
| **简化 (Simplification)** | 减少面片数量 |

### 3. 布尔运算修复
- 对非流形区域做切割
- 用简单体素填补空洞
- 然后做布尔并运算

## 基于学习的修复方法

### 传统方法的局限
- 需要人工参数调整
- 对复杂缺陷效果差
- 无法学习"正确"的几何

### 深度学习修复

#### MeshCNN (2019)
- 用图卷积处理网格
- 学习网格修复任务
- 但需要大量标注数据

#### PointNet/PointNet++
- 处理点云数据
- 学习点云补全

#### SGNN (2022)
- 利用 Signed Distance Field
- 学习 SDF 修复

#### Score-based / Diffusion 修复
- 类似图像修复
- 在几何空间去噪
- 填充缺失区域

## 重要研究方向

### 1. 自动缺陷检测
```
输入网格 ──→ 缺陷检测 ──→ 输出缺陷位置/类型
```

### 2. 端到端修复
```
输入缺陷网格 ──→ 神经网络 ──→ 输出修复网格
```

### 3. 可控修复
```
输入缺陷网格 + 约束 ──→ 满足约束的修复
```

### 4. 生成式修复
```
输入缺陷网格 ──→ 扩散模型 ──→ 合理完整网格
```

## 评估指标

| 指标 | 说明 |
|------|------|
| **水密性通过率** | 修复后是否水密 |
| **几何相似度** | 与原始设计偏差 |
| ** Hausdorff 距离** | 最大偏差 |
| **Chamfer 距离** | 平均偏差 |
| **面片质量** | 最小角、最大角 |

## 在本项目中的位置

```
CAD 模型输入
      ↓
缺陷检测 (AI)
      ↓
智能修复 (AI)
      ↓
验证 (水密性检测)
      ↓
输出合格 CAD
```

## 研究趋势

1. **大规模预训练**：用海量 CAD 数据预训练
2. **多模态**：结合图像、文本、几何
3. **物理约束**：让修复结果满足物理规律
4. **实时修复**：交互式修复体验
